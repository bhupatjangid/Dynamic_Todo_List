# MY todo list

- This is a web application where can add or remove todo items and can select title of the page according to the requirement



## Features
1. Responsive UI
2. Landing page
3. Add Todo Items
4. Remove Todo Items
5. Make any title of the page according to the requirements.

## Technology Stack

1. Node.js
2. Express.js
3. Css
4. Bootstrap
5. Javascript
6. Jquery
7. MongoDB
